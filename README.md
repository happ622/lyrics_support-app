
# 🎵 作詞支援アプリ - 芸術動力学モデル

このアプリは、芸術作品が人に与える印象を数理モデルで定量化し、作詞や構成を支援するツールです。

## 機能
- 感情の時間構成 g(t) をスライダーで調整
- 歌詞構造をエディタに記入
- 印象曲線 E(t) をグラフで可視化
- テンプレート方程式 aE'' - bE' + cEⁿ = S(t) に基づく

## 使用方法

```bash
pip install -r requirements.txt
streamlit run app.py
```
